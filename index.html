VecMap

TopoJSON形式の数値地図をSVGに変換して指定された要素内に出力するJavaScriptライブラリです。Webサイト上で日本や世界の白地図（国や地域などの輪郭線のみが描かれた地図）が手軽に使えます。

「Webサイト上に白地図を表示して全国の拠点をマーキングしたい」
「地図上のマーカーをホバーで変化させたり、属性でフィルタリングしたり、クリックでモーダルを出したりしたい」
「しかも、お客さんがその地図を自分で更新したいといっている」

という割とよくあるけれど頭の痛い要望に応えるために作ったライブラリです。

※本ライブラリはまだプロトタイプの段階です。仕様がコロコロ変わりますのでご注意ください。ご用命は柏井まで。

主な機能

  • 緯度と経度で指定した位置を中心に白地図を描画できます
  • 白地図を任意のスケールで拡大縮小して表示できます
  • SVGなので拡大縮小しても劣化しません
  • SVGなので地図の表示をCSSやJSで制御できます
  • 緯度と経度で指定した任意の位置にSVG形式のマーカーを置けます
  • 地図上のマーカーにクリックイベントを指定できます
  • TopoJSON中の任意のオブジェクトをキーワードで抽出して表示できます（例えば全国データから任意の都道府県のみ抽出して表示するなど）

制限

  • 街区レベルのナビゲーションに使えるような詳細度はありません（国・都道府県レベルまで）
  • IEでは動きません（babelを通すかポリフィル入れれば動くかも）
  • ローカル（file://〜）では動きません（少し改造すれば動くかも）。

Sample

日本全国(パラメータ全部入り)


const attr_japan = {
  map:"japan", //マップの種類を指定（デフォルトのマップを使用時）
  detail:"mid", //マップの詳細度を指定（デフォルトのマップを使用時）
  /*
  // TopoJSONファイルを直指定する場合は、以下のようにパスと読み込むオブジェクトのキーを指定します
  topojson:{
    file:'./assets/map/japan.mid.json', // TopoJsonファイルへのパス
    object:"japan" //表示するオブジェクト
  },
  */
  element:document.querySelector('.map.japan'), // 地図を出力する要素
  width:800, // 地図の幅(px)
  height:800, // 地図の高さ(px)
  // scale:1.0, // 拡大率、1.0で要素にデータ全体が収まるサイズになります。省略すると1.0。
  // center:[137.0, 38.2], // 地図の中心 [経度,緯度]。省略すると要素にデータ全体が収まる位置に調整されます 
  // offset:{x:0,y:0}, // centerで指定した位置を要素の中心からpx単位でオフセットします
  responsive:true, // true にすると地図のSVGのwidthとhightが省略され、外側の要素に合わせてサイズが可変します
  /*
  filter:{
    key:"name_ja", // TopoJson中のパラメータでデータをフィルタリングします。
    value:'東京都', // これは、全国の都道府県を含むデータから東京都だけを表示する指定です。
  },
  */
  marker:{
    image:'./assets/img/map_marker.svg',
    offset:{x:-8,y:-14},
    scale:1.2,
    list:[
      {
        id:'tokyo',
        class:'marker tokyo'
        latitude:'35.6754121',
        longitude:'139.5369406', 
        image:'./assets/img/home.svg',
        offset:{x:0,y:5},
        scale:1.5
      },
      {id:'nagoya',latitude:'35.1650768',longitude:'136.8678487'},
      {id:'asahikawa',latitude:'43.80926',longitude:'142.56692'},
      {id:'okinawa',latitude:'26.694338',longitude:'127.8780131'},
    ],
    click:(event) => {
      alert(event.currentTarget.id);
    }
  }
}
const map_japan = new VecMap(attr_japan);
map_japan.init()

北海道(フィルタリング)

地図データ（TopoJSON）中のサブカテゴリ（ここでは都道府県）を、キーワードでフィルタして表示します。scale=1.0で抽出された領域が要素にぴったり収まるように表示します。


const attr_hokkaido = {
  map:"japan",
  detail:"high",
  element:document.querySelector('.map.hokkaido'),
  width:800,
  height:800,
  responsive:true,
  filter:{
    key:"name_ja",
    value:'北海道',
  },
  marker:{
    image:'./assets/img/map_marker.svg',
    offset:{x:-8,y:-14},
    scale:1.2,
    list:[
      {id:'asahikawa',latitude:'43.80926',longitude:'142.56692'},
    ]
  }
}
const map_hokkaido = new VecMap(attr_hokkaido);
map_hokkaido.init();

東京都(フィルタリング＆ズーム)

東京都は、データ上では南は沖ノ鳥島、東は南鳥島まで含まれるため、デフォルトのスケールでは広大な領域が表示されてしまいます。ここではcenterとscaleを指定して、本州部分だけを表示しています。


const attr_tokyo = {
  map:"japan",
  detail:"high",
  element:document.querySelector('.map.tokyo'),
  width:800,
  height:400,
  scale:20.0,
  center:[139.5369406, 35.6754121],
  responsive:true,
  filter:{
    key:"name_ja",
    value:'東京都',
  },
  marker:{
    image:'./assets/img/map_marker.svg',
    offset:{x:-8,y:-14},
    scale:1.2,
    list:[
      {
        id:'tokyo',
        class:'marker tokyo',
        latitude:'35.6754121',
        longitude:'139.5369406', 
      },
    ]
  }
}
const map_tokyo = new VecMap(attr_tokyo);
map_tokyo.init()

正射図法（Orthographic）

応用

二つ地図を使って南西諸島をサブマップに表示し、地図表示後に実行されるhookを使って各都道府県クリックで拡大地図が出るようにしてあります。

また、マーカーについてはHTMLのdata属性から取得して地図にプロットしています。この方法を使うと、JSやデータファイルを修正しなくても、HTMLを更新すれば地図も更新されます。カスタムブロックなどの機能を使ってデータ属性を修正可能にしておけばWPから地図を更新することもできるはず。

  const data_list = adv_map_list.querySelectorAll('.advanced > .list > .list__item');
  const location_list = [];
  data_list.forEach((element,index) => {
    let data_location = element.dataset.location.split(',')

  const attr_adv = {
    main:{
      map:"japan",
      detail:"mid",
      element:document.querySelector('.advanced .map .main'),
      width:1200,
      height:1200,

      center:[137.0, 38.2],
      responsive:true,
      marker:{

  const adv_map_main = new VecMap(attr_adv.main);

  adv_map_main.post_map_load_hook(function(){
    const adv_region = document.querySelectorAll('.advanced>.map .region');
    let modal_map, modal_map_conf;
    adv_region.forEach((element,index) => {
      element.addEventListener('click',(e)=>{
        const region_id = e.currentTarget.dataset.id;
        modal_map_conf = {
          map:"japan",
          detail:"mid",
          element:document.querySelector('.advanced>.map>.modal>.modal_map'),
          width:400,
          height:400,
          responsive:true,
          filter:{
            key:"id",
            value:region_id
          },
          marker:{
          image:'./assets/img/map_marker.svg',
          offset:{x:-8,y:-14},
          scale:1.5,
          list:location_list,
            click:(event) => {
              alert(event.currentTarget.id)
            }
          }
        }
        modal_map = new VecMap(modal_map_conf);
        modal_map.init();
        document.querySelector('.advanced .modal').classList.add('is-open')
        document.querySelector('.advanced .modal .modal_title').innerHTML = e.currentTarget.dataset.name_ja;
      })
    })
    document.querySelector('.advanced .modal_close').addEventListener('click',(e) => {
      document.querySelector('.advanced .modal_map svg').remove();
      document.querySelector('.advanced .modal').classList.remove('is-open');
    })
  })

  adv_map_main.init()

  const adv_map_sub = new VecMap(attr_adv.sub);
  adv_map_sub.init()
